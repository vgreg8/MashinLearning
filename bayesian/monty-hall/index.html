Задача Монти Холла

Материалы > Байесовская статистика

Рассмотрим задачу Монти Холла (Monty Hall problem).

Условие задачи

В телевизионной игре участнику предлагается отгадать, за какой из трех дверей находится выигрыш — автомобиль (car, C). По условию за каждой из двух оставшихся дверей спрятано по козе (goat, G).

На первом шаге участнику необходимо выбрать (но не открыть) одну из трех дверей. На втором — ведущий (его зовут Монти Холл) открывает одну из двух оставшихся дверей, за которой находится коза.

Вопрос: после того как ведущий открыл дверь с козой, имеет ли смысл участнику изменить свой изначальный выбор или после замены выбранной двери вероятность выигрыша останется той же?

Важно! Ведущий всегда откроет ту дверь, за которой находится коза, а не приз.

Интуитивное объяснение

Всего возможны два сценария:

  • Сценарий А. Участник изначально выбирает дверь с призом и тогда правильной стратегией будет сохранить свой выбор и не менять дверь;
  • Сценарий Б. Участник выбирает дверь с козой и тогда (поскольку ведущий покажет, где находится вторая коза) выигрышной стратегией будет смена двери.

При этом очевидно, что в двух случаях из трех реализуется сценарий Б. Как следствие, в двух случаях из трех выгоднее сменить дверь, а значит вероятность выигрыша при смене равна $\tfrac{2}{3}.$

Аналитическое решение

Поскольку все три двери одинаковы, вероятность выбрать одну из них участником равна $\tfrac{1}{3}.$

Далее для определенности предположим, что участник выбрал первую дверь и автомобиль находится именно за этой дверью, $d_1=C.$ В этом случае ведущий с одинаковой вероятностью, равной $\tfrac{1}{2},$ откроет либо вторую, $open = d_2,$ либо третью, $open = d_3,$ дверь.

Если же участник выбрал вторую или третью дверь, за которыми находятся козы, то ведущему по условию задачи ничего не остается, кроме как открыть дверь с другой козой.

Таким образом,

  • если участник сохранит свой выбор, вероятность выигрыша будет равна $\tfrac{1}{6} + \tfrac{1}{6} = \tfrac{1}{3};$
  • если же он решит сменить дверь, то $0 + \tfrac{1}{3} + 0 + \tfrac{1}{3} = \tfrac{2}{3}.$

Примечание. Если ведущий «забудет», за какой из оставшихся дверей находится автомобиль, то тогда действительно, смена двери участником потеряет смысл.

Байесовский подход

К решению этой задачи можно подойти с точки зрения теоремы Байеса.

По-прежнему выберем первую дверь и предположим без потери общности, что ведущий открыл третью дверь, за которой по условию должна находиться коза. Как следствие, автомобиль может быть либо за первой дверью, либо за второй.

Вероятность выигрыша при стратегии «остаться» в этом случае можно сформулировать как

$$ P(d_1 = C \mid open = d_3), $$

то есть это вероятность того, что машина находится за первой дверью при условии, что ведущий открыл третью.

Вероятность выигрыша при смене двери с первой на вторую, таким образом, можно представить как

$$ P(d_2 = C \mid open = d_3). $$

По формуле Байеса,

$$ \begin{split} P(d_1=C \mid open = d_3) &= \tfrac{ P(open = d_3 \mid d_1 = C) P(d_1 = C) }{P (open = d_3)} \\[1ex] &= \tfrac{ P(open = d_3 \mid d_1 = C) P(d_1 = C) }{P (open = d_3 \mid d_1 = C) P(d_1 = C) + P (open = d_3 \mid d_2 = C) P(d_2 = C) } \\[1ex] &= \tfrac{ \tfrac{1}{2} \cdot \tfrac{1}{3} }{ \tfrac{1}{2} \cdot \tfrac{1}{3} + 1 \cdot \tfrac{1}{3} } = \frac{1}{3} \end{split} $$

$$ \begin{split} P(d_2=C \mid open = d_3) &= \tfrac{ P(open = d_3 \mid d_2 = C) P(d_2 = C) }{P (open = d_3)} \\[1ex] &= \tfrac{ P(open = d_3 \mid d_2 = C) P(d_2 = C) }{P (open = d_3 \mid d_1 = C) P(d_1 = C) + P (open = d_3 \mid d_2 = C) P(d_2 = C) } \\[1ex] &= \tfrac{ 1 \cdot \tfrac{1}{3} }{ \tfrac{1}{2} \cdot \tfrac{1}{3} + 1 \cdot \tfrac{1}{3} } = \frac{2}{3} \end{split} $$

Другими словами, в соответствии с терминологией байесовской статистики, новая информация (открытие третьей двери) или правдоподобие позволило обновить наше априорное представление о том, что каждая из дверей равновероятна.
